### Objectif

Dans cet exercice, vous allez apprendre à :

- structurer un montage audio dans REAPER
- intégrer une ambiance sonore (sound design simple)
- améliorer la clarté d’un récit
- utiliser les bases du mix (volume, fade, organisation des pistes)


## Mise en contexte

Vous travaillez sur un **court récit audio (1 à 3 minutes)**.

Votre mission :

👉 Transformer un enregistrement brut en **histoire immersive** avec ambiance.


# Matériel fourni

Vous aurez :

- 1 narration (voix principale)
- 1 ambiance sonore (ex : pluie, café, rue, forêt)
- 1 ou 2 effets sonores (optionnel)


# Étape 1 – Organisation du projet (REAPER)

1. Créer un nouveau projet
2. Nommer le projet :
```
Nom_Prenom_Exercice2_Reaper
```

3. Importer les fichiers audio
4. Organiser les pistes :

- Piste 1 : Narration
- Piste 2 : Ambiance
- Piste 3 : Effets (si applicable)


# Étape 2 – Nettoyage de la narration

Travail sur la voix :

- supprimer les silences inutiles
- couper les erreurs
- garder un rythme naturel

### Important :

Ne pas rendre la voix robotique.



# Étape 3 – Ajout de l’ambiance sonore

Ajouter une ambiance en arrière-plan.

### Consignes :

- volume très bas (-18 dB à -30 dB)
- loop si nécessaire
- éviter de couvrir la voix



# Étape 4 – Équilibre des volumes

### Objectif :

Créer une hiérarchie claire :

- Voix = premier plan
- Ambiance = arrière-plan
- Effets = ponctuels

### À vérifier :

- la voix est toujours compréhensible
- aucune piste ne domine inutilement


# Étape 5 – Fondu et transitions

Ajouter :

- fade in sur l’ambiance
- fade out à la fin
- transitions douces entre sections


# Étape 6 – Mini mix

Appliquer des ajustements simples :

- baisse de volume si saturation
- léger EQ si nécessaire (optionnel)
- panoramique léger pour élargir l’espace


# Étape 7 – Exportation

Exporter votre projet :

### Format :

- WAV (qualité maximale)
ou
- MP3 (si demandé)

### Paramètres :

- 44.1 kHz ou 48 kHz
- Stereo
- Normalize OFF (sauf indication contraire)

### Nom du fichier :
```
Nom_Prenom_Exercice2.wav
```
